What is this?

This survey aims to collect and visualize data about how engineering students were accepted into their streams. Your submission is anonymous and will help us analyze GPA distributions, first-choice stats, and more, so we can understand stream competitiveness and selection trends.

This site was made using HTML/CSS, JS, and Python w/ SQLite and Flask. It is hosted via Github Pages and Render.
